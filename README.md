#### 使用说明

开始启动程序，前提条件需要先开启zookeeper服务注册中心。 

启动顺序：先启动ProviderTest测试类，也就是启动服务提供者，将服务暴露出去，然后启动ConsumerTest测试类，也就是启动服务消费者。 

为了方便观察，我们可以将dubbo-admin也启动起来，这样方便观察服务。
